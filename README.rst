*********************************************
OIPA_auth: Authentication middleware for OIPA
*********************************************
============
Installation
============

*Clone this repository

*Run the install script

*add the middleware to settings.py

*add OIPA_auth_settings to settings.py this should look someting like:
.. code-block:: python
    OIPA_AUTH_SETTINGS = {
        'HOST': 'localhost',
        'PORT': '8080',
    }
